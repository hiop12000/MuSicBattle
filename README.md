паузы:

13 только шестнадцатая (продливать написанием спец знаков)

_%% шестнадлцатая
_%  восьмая
_   четвертная пауза
_*  половинная
_** целая

аналогично с нотами:

1%%  шестнадцатая
1%  восьмая
1  четвертная
1* половинная
1**  целая

(в примере выше используется нота до)

используется логика:

% - делить на 2
* - умножить на 2

пользователь вводит последовательность, которая переводится в массив строк, он впоследствии сверяется
